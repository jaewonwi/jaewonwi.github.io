Backend

  • Java객체지향 설계와 예외 처리 중심의 서비스 로직 구현

  • JPA도메인 모델링, 연관관계, 트랜잭션 관리

Database

  • MySQL테이블 설계, 인덱스, 실행 계획 확인

  • Redis캐싱과 만료 데이터 관리

Infra

  • Docker로컬과 배포 환경 차이를 줄이는 컨테이너 구성
  • AWSEC2, RDS 기반의 기본 배포 환경 구성
  • GitHub Actions테스트와 배포 자동화 흐름 구성

Tools
